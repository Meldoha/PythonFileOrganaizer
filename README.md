# Python File Organizer

Скрипт для автоматической сортировки и переименования файлов в папке.

## Возможности
- Сортировка файлов по расширениям
- Автоматическое создание подпапок
- Переименование файлов по заданному шаблону
- Обработка ошибок (несуществующий путь, пустая папка)

## Требования
- Python 3.8+

## Установка и запуск
1. Склонировать репозиторий или скачать архив
2. Перейти в папку с проектом
3. Запустить команду:

python main.py "/path/to/folder" - Путь к папке (без ковычек)

## Пример
До запуска:
- file1.txt
- image1.png
- doc1.pdf

После запуска:
- TXT/file_1.txt
- PNG/image_1.png
- PDF/doc_1.pdf
